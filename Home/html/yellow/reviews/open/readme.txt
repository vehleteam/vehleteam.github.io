OrDomain is a Responsive HTML5 WHMCS Hosting Template designed for All kinds of Domain and Hosting Business. It's an HTML template based on Bootstrap v3.3.7. Also included WHMCS Template based on WHMCS v7.7. Anyone can easily update/edit this template to follow our Well Sorted Documentation.
